# quasar-babel-test

## Project setup
```
npm uninstall -g vue-cli
npm install -g @vue/cli@3.12.1
vue create quasar-babel-test
cd quasar-babel-test/
npm i vue-cli-plugin-quasar@0.17.1 --save
vue add quasar
npm run serve
npm install
```
